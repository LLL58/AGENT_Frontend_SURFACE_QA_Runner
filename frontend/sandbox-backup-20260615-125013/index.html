功能特性

页面健康检查

检测页面是否正常加载，是否有白屏、JavaScript 错误等问题。

控件扫描

自动扫描页面上的可交互控件，如按钮、链接、输入框等。

错误收集

收集页面运行时的各种错误，包括 JavaScript 错误、网络请求错误等。
